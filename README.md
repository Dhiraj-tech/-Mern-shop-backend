# mern-shop-ecommerce

mern-shop-ecommerce is a feature-rich e-commerce web application developed using the MERN stack (MongoDB, Express, React, and Node.js). This platform enables users to explore and purchase a wide range of products, exploring various categories and brands.

# Features:

- [ ] Secure user registration and login functionality.
- [ ] User account management with the ability to update profiles securely.
- [ ] Efficient admin dashboard to oversee product inventory, user data, and orders.
- [ ] Authentication is implemented using JSON Web Token (JWT) for enhanced protection.
- [ ] Visually appealing homepage with a hero banner showcasing popular products.
- [ ] Search bar for quick and efficient product discovery.
- [ ] Filtering options based on price, brand, category, and more for a personalized shopping experience.
- [ ] Detailed product pages with specifications, compatibility details, pricing, and customer reviews.
- [ ] Use high-quality images and concise product descriptions to provide essential information to users.
- [ ] Wishlist and favorites feature for users to save products for future reference.
- [ ] Allow users to easily add or remove items from the cart.
- [ ] Implement a smooth and secure checkout process.
- [ ] Allow customers to leave reviews and ratings for products
- [ ] Provide contact us form to the user.

# Technology used:

- [ ] MongoDB
- [ ] Express
- [ ] React
- [ ] Node.js
- [ ] JWT
